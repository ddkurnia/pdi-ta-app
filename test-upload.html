🔧 Test Cloudinary Upload

📋 Cara Cek Cloud Name

1. Buka https://console.cloudinary.com
2. Login / Sign Up
3. Di dashboard atas, ada tulisan:
Cloud name: XXXXXX
4. Copy cloud name tersebut, paste di atas

📋 Cara Buat Upload Preset

1. Buka **Settings** (icon gear ⚙️)
2. Pilih **Upload** tab
3. Scroll ke **Upload presets**
4. Klik **Add upload preset**
5. Isi:
  • Upload preset name: pdi-ta-upload
  • Signing mode: Unsigned
6. Klik **Save**

⚠️ Signing mode HARUS Unsigned
